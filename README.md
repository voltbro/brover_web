# brover_web — agro_reg2026

Ветка `agro_reg2026` содержит версию веб-интерфейса BRover-E5 для работы с
системой полива. Помимо базового управления ровером, интерфейс управляет
поворотом камеры, насосом и направлением полива через плату VBCores ServoBoard,
подключённую к `CAN1`.

## Отличия от main

- интерфейс рассчитан на одну видеокамеру;
- добавлено управление сервоприводами поворота и наклона камеры;
- добавлены два режима работы насоса: наполнение бака и полив;
- добавлено управление направлением полива;
- добавлен ROS 2-узел `irrigation_control`, преобразующий команды из ROS 2 в
  сообщения CAN FD;
- на телефоне доступны отдельные режимы «Агро» и «Движение».

## Возможности интерфейса

- просмотр видео с камеры ровера;
- управление движением с клавиатуры или экранного джойстика;
- ограничение максимальной скорости;
- отображение одометрии и напряжения аккумулятора;
- сброс одометрии;
- управление поворотом и наклоном камеры;
- управление насосом при удержании кнопки;
- пошаговое управление направлением полива;
- отображение состояния ROS и Wi-Fi;
- светлая и тёмная темы.

## Схема взаимодействия

```text
Браузер
 ├─ HTTP :8080 ─────→ Flask-сервер brover_web
 ├─ HTTP :9999 ─────→ web_video_server → видео с камеры
 └─ WebSocket :9090 → rosbridge_server → ROS 2
                                           │
                                           ├─ /cmd_vel → управление ровером
                                           └─ агро-топики
                                                ↓
                                     irrigation_control
                                                ↓
                                      CAN FD, CAN1, ID 0x100
                                                ↓
                                      VBCores ServoBoard
                                       ├─ сервоприводы камеры
                                       ├─ сервоприводы полива
                                       └─ насос
```

Веб-страница публикует команды в ROS 2 через `roslib.js`. Узел
`irrigation_control` принимает агро-команды, формирует четырёхбайтовое сообщение
CAN FD и отправляет его в интерфейс `can1`.

## Управление с компьютера

### Движение ровера

| Клавиша | Действие |
| --- | --- |
| `W` | движение вперёд |
| `S` | движение назад |
| `A` | поворот влево |
| `D` | поворот вправо |

Ползунок «Максимальная скорость» задаёт ограничение от 10 до 100 процентов.

### Направление полива

| Клавиша | Действие |
| --- | --- |
| `J` | поворот полива влево |
| `L` | поворот полива вправо |
| `K` | направление полива вверх |
| `I` | направление полива вниз |

Каждое нажатие изменяет соответствующий угол на 2 градуса. Эти же команды
доступны через кнопки со стрелками в панели «Система полива».

## Управление с телефона

На мобильном экране доступны два режима:

- **Агро** — камера, насос и кнопки изменения направления полива;
- **Движение** — экранный джойстик управления ровером.

Переключатель режимов расположен под видеопотоком. В нижней части экрана
показывается напряжение аккумулятора. Буквенные обозначения клавиш внутри кнопок
полива на телефоне скрыты, остаются только стрелки.

## Управление камерой

В интерфейсе используются два ползунка:

| Ползунок | Диапазон интерфейса | Значение для сервопривода |
| --- | --- | --- |
| Поворот | от `-90°` до `90°` | от `160°` до `20°` |
| Наклон | от `-45°` до `45°` | от `0°` до `150°` |

При открытии страницы публикуются исходные значения: поворот `90°`, наклон
`60°`. Команды передаются через топики `/camera/pan` и `/camera/tilt`.

## Управление насосом

Кнопки насоса работают только во время удержания.

| Кнопка | Значение | Действие выходов |
| --- | --- | --- |
| остановка после отпускания | `0` | `PB5 = LOW`, `PB7 = LOW` |
| «Наполнить бак» | `1` | `PB5 = LOW`, `PB7 = HIGH` |
| «Полив» | `2` | `PB5 = HIGH`, `PB7 = LOW` |

При нажатии команда публикуется в `/irrigation/pump`. При отпускании кнопки,
потере указателя или переключении вкладки публикуется состояние `0`.

## Управление направлением полива

Начальное положение механизма полива:

- поворот: `90°`;
- наклон: `160°`.

Допустимые диапазоны ограничиваются и в веб-интерфейсе, и в Arduino-скетче:

| Ось | Минимум | Максимум | Шаг интерфейса |
| --- | --- | --- | --- |
| поворот | `40°` | `180°` | `2°` |
| наклон | `94°` | `160°` | `2°` |

## Подключение агрооборудования

Raspberry Pi ровера соединён с VBCores ServoBoard через `CAN1`. Исполнительные
устройства подключены к ServoBoard следующим образом:

| Устройство | Назначение | Контакт в рабочем скетче |
| --- | --- | --- |
| сервопривод камеры | поворот | `PA0` |
| сервопривод камеры | наклон | `PA1` |
| сервопривод полива | поворот | `PB1` |
| сервопривод полива | наклон | `PB0` |
| насос, клеммы `OUT` разъёма `M2` | направление 1 | `PB5` |
| насос, клеммы `OUT` разъёма `M2` | направление 2 | `PB7` |

Документация платы:
[docs.vbcores.ru/docs/Hardware/servo-board](https://docs.vbcores.ru/docs/Hardware/servo-board).

## Arduino-скетч

Скетч для ServoBoard находится в файле
[`Arduino/agro_reg2026/agro_reg2026.ino`](Arduino/agro_reg2026/agro_reg2026.ino).

Он выполняет следующие задачи:

- инициализирует CAN FD;
- принимает сообщения из FIFO0;
- определяет устройство и команду по первым двум байтам;
- задаёт углы четырёх сервоприводов;
- управляет двумя выходами насоса.

## Формат CAN-команд

Узел `irrigation_control` отправляет расширенные сообщения CAN FD с
идентификатором `0x100`.

Полезная нагрузка состоит из четырёх байтов:

| Байт | Содержимое |
| --- | --- |
| `0` | идентификатор устройства |
| `1` | идентификатор команды |
| `2` | значение команды |
| `3` | `0x00`, зарезервирован |

Идентификаторы устройств:

| Устройство | ID |
| --- | --- |
| камера | `0x01` |
| механизм полива | `0x02` |
| насос | `0x03` |

Идентификаторы команд:

| Команда | ID |
| --- | --- |
| наклон | `0x01` |
| поворот | `0x02` |
| состояние насоса | `0x03` |

## ROS 2-топики

| Направление относительно браузера | Имя | Тип | Назначение |
| --- | --- | --- | --- |
| публикация | `/cmd_vel` | `geometry_msgs/msg/Twist` | управление движением |
| публикация | `/camera/pan` | `std_msgs/msg/Int16` | поворот камеры |
| публикация | `/camera/tilt` | `std_msgs/msg/Int16` | наклон камеры |
| публикация | `/irrigation/nozzle/pan` | `std_msgs/msg/Int16` | поворот полива |
| публикация | `/irrigation/nozzle/tilt` | `std_msgs/msg/Int16` | наклон полива |
| публикация | `/irrigation/pump` | `std_msgs/msg/Int16` | режим насоса |
| подписка | `/odom_pose2d` | `geometry_msgs/msg/Pose2D` | координаты и угол ровера |
| подписка | `/bat` | `sensor_msgs/msg/BatteryState` | напряжение аккумулятора |
| сервис | `/odom/reset` | `std_srvs/srv/Empty` | сброс одометрии |

Узел `irrigation_control` подписывается на пять агро-топиков и передаёт их
значения в `CAN1`.
